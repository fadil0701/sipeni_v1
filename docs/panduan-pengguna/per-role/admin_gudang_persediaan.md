# Panduan: Admin Gudang Persediaan

> **Akses di aplikasi:** Menu sidebar **Panduan Pengguna** — bab ini otomatis ditampilkan untuk role Anda.

## Profil role

| | |
|---|---|
| **Level** | Pusat |
| **Modul default** | Inventory, Transaksi |
| **Kategori gudang** | PERSEDIAAN saja |

## Kegunaan

Mengelola gudang **barang persediaan** (ATK, alkes non-farmasi, dll.) — stok, disposisi, dan distribusi.

## Menu utama

| Menu | Kegunaan |
|------|----------|
| **Inventory** | Data inventory, stock, opname, kartu stok |
| **Distribusi → Daftar Permintaan** | Proses draft distribusi persediaan |
| **Distribusi → SBBK** | Dukung proses/kirim (jika diizinkan) |

## Alur harian

1. Cek permintaan disposisi **persediaan** masuk
2. Verifikasi **stok** di Data Stock
3. **Daftar Permintaan** → Proses → buat **SBBK** per permintaan
4. **Kirim** SBBK dari menu Distribusi Barang (SBBK)
5. **Opname** berkala → stock adjustment

## Tips

- Permintaan unit filter jenis **Persediaan** — jangan campur farmasi
- Monitor item slow-moving via kartu stok
