# Panduan: Pengurus Barang

> **Akses di aplikasi:** Menu sidebar **Panduan Pengguna** — bab ini otomatis ditampilkan untuk role Anda.

## Profil role

| | |
|---|---|
| **Level** | Pusat |
| **Modul default** | Inventory, Transaksi, Aset |

## Kegunaan

**Pengelola operasional persediaan** — disposisi permintaan, compile **SBBK**, distribusi, oversight seluruh gudang pusat.

## Menu utama

| Menu | Kegunaan |
|------|----------|
| **Approval** | **Disposisi** permintaan ke gudang kategori |
| **Distribusi → Daftar Permintaan** | Monitor draft distribusi |
| **Distribusi → SBBK** | Compile, proses, kirim, cetak |
| **Inventory** | Stok pusat, kartu stok, opname |
| **Aset** | Register, KIR, mutasi |

## Alur permintaan (setelah approve kepala pusat)

1. **Approval** → **Disposisi** → pilih gudang kategori (Persediaan/Farmasi/Aset)
2. Monitor **Daftar Permintaan** — admin gudang kategori proses item
3. **Compile SBBK** (gabung permintaan → satu surat keluar)
4. **Distribusi SBBK** → proses → **kirim** → cetak PDF
5. Unit **penerimaan** → status selesai

## Tips

- Prioritaskan permintaan CITO
- Cek stok pusat sebelum disposisi — jika kosong, koordinasi pengadaan
- Compile SBBK per unit/tanggal untuk efisiensi pengiriman
