# Panduan: Kepala Unit

> **Akses di aplikasi:** Menu sidebar **Panduan Pengguna** — bab ini otomatis ditampilkan untuk role Anda.

## Profil role

| | |
|---|---|
| **Level** | Unit |
| **Scope** | Hanya unit kerja sendiri |
| **Modul default** | Transaksi, Inventory, Aset, Planning |

## Kegunaan

Pimpinan unit — **mengetahui** permintaan unit, **menerima** barang distribusi, oversight stok & aset unit.

## Menu utama

| Menu | Kegunaan |
|------|----------|
| **Transaksi → Permintaan Barang** | Lihat/monitor permintaan unit (bisa ajukan jika diizinkan) |
| **Approval** | **Mengetahui** permintaan unit |
| **Distribusi → Penerimaan Barang** | Verifikasi barang diterima |
| **Inventory → Data Stock** | Stok unit |
| **Aset → Register Aset** | Oversight aset unit |

## Langkah: Mengetahui permintaan

1. Approval → pilih permintaan unit Anda
2. Detail → cek kebutuhan
3. Klik **Mengetahui**

## Langkah: Penerimaan barang

1. Distribusi → **Penerimaan Barang**
2. Pilih distribusi status **dikirim**
3. Verifikasi qty fisik vs SBBK
4. **Verifikasi penerimaan** → stok unit terupdate

## Tips

- Pastikan admin unit sudah input permintaan lengkap sebelum Anda mengetahui
- Retur barang rusak via menu Retur jika ada cacat saat terima
