# Panduan: Admin Gudang Aset

> **Akses di aplikasi:** Menu sidebar **Panduan Pengguna** — bab ini otomatis ditampilkan untuk role Anda.

## Profil role

| | |
|---|---|
| **Level** | Pusat |
| **Modul default** | Inventory, Aset |
| **Kategori gudang** | ASET saja |

## Kegunaan

Mengelola gudang pusat **kategori aset** — inventory aset tetap, register aset, proses disposisi permintaan aset.

## Menu utama

| Menu | Kegunaan |
|------|----------|
| **Inventory → Data Inventory/Stock** | Stok aset di gudang |
| **Aset → Register Aset** | Pencatatan register & rincian |
| **Aset → KIR / Mutasi** | Dokumen & perpindahan |
| **Distribusi → Daftar Permintaan** | Proses disposisi item aset |

## Alur disposisi permintaan aset

1. Terima disposisi dari pengurus barang (kategori aset)
2. Daftar Permintaan → **Proses** → buat **SBBK**
3. Isi qty real dari stok gudang aset
4. Simpan SBBK, lalu **kirim** dari menu Distribusi Barang (SBBK)

## Tips

- Register aset segera setelah penerimaan pengadaan
- Mutasi aset jika barang langsung dipindah ke unit/ruangan
