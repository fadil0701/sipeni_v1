# Panduan: Kepala Pusat

> **Akses di aplikasi:** Menu sidebar **Panduan Pengguna** — bab ini otomatis ditampilkan untuk role Anda.

## Profil role

| | |
|---|---|
| **Level** | Pusat |
| **Scope** | Lintas unit (monitoring & approval) |
| **Modul default** | Transaksi, Laporan |

## Kegunaan

Pimpinan tertinggi operasional — **persetujuan akhir** permintaan barang dan **RKU**, serta monitoring laporan.

## Menu utama

| Menu | Kegunaan |
|------|----------|
| **Approval** | Approve / reject permintaan barang |
| **Transaksi → Daftar RKU** | Review & approve RKU |
| **Monitoring** | Laporan ringkasan & stok gudang |
| **Dashboard** | Overview |

## Alur kerja Anda

### Permintaan barang

1. Buka **Approval** → filter status menunggu kepala pusat
2. Buka **Detail** → periksa item, qty, keterangan unit
3. Pilih **Setujui** (lanjut disposisi gudang) atau **Tolak** (berhenti)

### RKU

1. Transaksi / Perencanaan → **Daftar RKU**
2. Buka RKU status review kepala pusat
3. **Setujui** / **Tolak** / minta **Revisi**

## Tips

- Koordinasi dengan Kasubbag TU sebelum reject — pastikan alasan jelas
- Gunakan laporan stok gudang untuk validasi ketersediaan sebelum approve volume besar
