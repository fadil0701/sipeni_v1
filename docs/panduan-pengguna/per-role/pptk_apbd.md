# Panduan: PPTK APBD

> **Akses di aplikasi:** Menu sidebar **Panduan Pengguna** — bab ini otomatis ditampilkan untuk role Anda.

## Profil role

| | |
|---|---|
| **Level** | Pusat |
| **Modul default** | Planning, Procurement, Reports |
| **Legacy setara** | `admin_pptk_apbd` |

## Kegunaan

**Pelaksana teknis kegiatan APBD** — monitoring RKU, pengadaan, dan laporan untuk sumber anggaran APBD.

## Menu utama

| Menu | Kegunaan |
|------|----------|
| **Perencanaan** | Program, RKU, Rekap |
| **Pengadaan** | Paket & proses (monitoring) |
| **Monitoring** | Laporan stok & ringkasan |

## Tugas

1. Monitor RKU & paket pengadaan dengan sumber **APBD**
2. Validasi kesesuaian kegiatan dengan DPA
3. Review laporan realisasi vs rencana
4. Koordinasi dengan pengadaan & keuangan

## Tips

- Filter data berdasarkan sumber anggaran APBD di master
- Gunakan rekap tahunan untuk pelaporan ke pimpinan
