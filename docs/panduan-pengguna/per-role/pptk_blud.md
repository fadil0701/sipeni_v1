# Panduan: PPTK BLUD

> **Akses di aplikasi:** Menu sidebar **Panduan Pengguna** — bab ini otomatis ditampilkan untuk role Anda.

## Profil role

| | |
|---|---|
| **Level** | Pusat |
| **Modul default** | Planning, Procurement, Reports |
| **Legacy setara** | `admin_pptk_blud` |

## Kegunaan

**Pelaksana teknis kegiatan BLUD** — monitoring RKU, pengadaan, dan laporan untuk sumber anggaran BLUD.

## Menu utama

| Menu | Kegunaan |
|------|----------|
| **Perencanaan** | Program, RKU, Rekap |
| **Pengadaan** | Paket & proses (monitoring) |
| **Monitoring** | Laporan |

## Tugas

1. Monitor RKU & pengadaan sumber **BLUD**
2. Pastikan kegiatan sesuai rencana bisnis BLUD
3. Koordinasi realisasi dengan pengadaan & keuangan
4. Sediakan data untuk laporan BLUD

## Perbedaan dengan PPTK APBD

Fokus **sumber anggaran BLUD** — alur sistem sama, filter kegiatan/anggaran berbeda.

Lihat juga: [pptk_apbd.md](./pptk_apbd.md)
